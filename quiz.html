Studietaak 4 Toestand en repetitie in code

Deze studietaak is speciaal gemaakt voor studenten die forensische ICT volgen en net beginnen met programmeren. Je leert hoe je herhalende taken kunt automatiseren, bijvoorbeeld het meerdere keren uitvoeren van dezelfde opdracht of het doorlopen van een verzameling gegevens.

Door eenvoudige voorbeelden ontdek je hoe je met lussen (zoals een for-lus of while-lus) overzichtelijke en herbruikbare code kunt schrijven. Deze kennis helpt bij het analyseren van digitale sporen en het verwerken van data in een forensische context.

Leerdoelen

  1. Ik kan uitleggen wat repetitie is in code en waarom dit nodig is.
  2. Ik kan uitleggen wat een lijstdatatype is en waarom dit nodig is.
  3. Ik ben in staat om een probleem met iteraties kleiner te maken.
  4. Ik ben in staat om gestructureerd een probleem met iteraties te analyseren.
  5. Ik kan in code het lijstdatatype gebruiken.
  6. Ik kan lussen in code gebruiken, waaronder:
    • while-lus
    • for-lus
    • break
    • continue
